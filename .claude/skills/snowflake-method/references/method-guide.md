# 雪花写作法参考指南

> 基于兰迪·英格曼森（Randy Ingermanson）所著《雪花写作法：10步写出一篇好小说》

## 什么是雪花写作法？

雪花写作法的灵感来自数学中的分形——就像雪花从一个简单的三角形开始，每一步都在边上增加更多的三角形，最终形成复杂而美丽的雪花图案。写小说也可以这样：从一个简单的想法出发，通过10个渐进步骤，逐步扩展成一部完整的小说。

**核心哲学：** 不要试图一步到位。好的小说是"设计"出来的，不是一蹴而就的。

## 三幕结构

雪花写作法基于经典的三幕结构：

```
第一幕（约25%）        第二幕（约50%）           第三幕（约25%）
[铺垫 → 第一次灾难]   [对抗 → 第二次灾难 →      [高潮 → 结局]
                       第三次灾难]
```

### 三次灾难的规律

1. **第一次灾难**：推动主角从舒适区走出，进入故事主线。通常由外部力量引起。
2. **第二次灾难**：情况进一步恶化，stakes升高。同样由外部原因导致。
3. **第三次灾难**：最黑暗的时刻。关键区别——这次灾难是由主角自身的弱点、过失或内在缺陷导致的。这迫使主角面对自己的问题，从而获得成长。

## 场景的两种类型

每个场景要么是"主动场景"，要么是"被动场景"，两者交替出现：

### 主动场景（Proactive Scene）

角色主动追求目标的场景。包含三个要素：

- **目标（Goal）**：视角人物在这个场景中想要达成的具体事情
- **冲突（Conflict）**：阻碍目标达成的障碍
- **挫折（Setback）**：场景结束时情况变得更糟（目标未达成，或达成了但付出了意外代价）

### 被动场景（Reactive Scene）

角色应对挫折的场景。包含三个要素：

- **反应（Reaction）**：角色对上一场景挫折的情感回应
- **困境（Dilemma）**：角色面临两难选择——两个选项都不好
- **决定（Decision）**：角色做出选择，设定新目标，引向下一个主动场景

### 场景交替模式

```
主动场景 → 被动场景 → 主动场景 → 被动场景 → ...
(目标→冲突→挫折) → (反应→困境→决定) → (目标→冲突→挫折) → ...
```

这种交替节奏让故事既有紧张的行动，又有深入的情感，形成自然的呼吸感。

## 人物塑造的三个层次

1. **价值观**（抽象层面）：对这个人来说，什么是最重要的？ — "没有什么比___更重要"
2. **目标**（具体层面）：他/她具体想要达成什么？
3. **冲突**（戏剧张力）：价值观和目标之间的张力，角色和外部世界的碰撞

**关键原则：** 角色的价值观和目标之间应该存在内在矛盾。比如一个重视家庭的人，却必须离开家人去执行危险任务——这种张力产生戏剧性。

## 顿悟（Epiphany）

每个主要角色在故事结束时应该有一个"顿悟时刻"：

- 角色认识到自己之前的某些信念是错误的
- 这个领悟改变了角色的行为方式
- 顿悟应该从故事事件中自然产生，而非突然出现
- 不是所有角色都会正面顿悟——反派可能拒绝改变

## 情节与人物的关系

雪花写作法的一个核心洞见是：**情节和人物是一枚硬币的两面**。

- 好的情节源自人物的选择和行动
- 人物在情节的压力下才会展现真实的自我
- 步骤在情节扩展（步骤2→4→6）和人物深化（步骤3→5→7）之间交替进行

## 实用建议

### 关于修改
- 雪花法的每一步都可能导致你回去修改前面的步骤——这是正常的
- 越早发现问题越好，修改大纲比修改10万字的小说容易得多
- 不必严格按顺序走完所有步骤，可以根据需要灵活调整

### 关于第一稿
- 有了前九步的充分准备，写第一稿的速度会大大加快
- 第一稿不必完美，重要的是把故事写出来
- 写的过程中可能产生新的灵感，不必害怕偏离大纲

### 关于篇幅参考
- 步骤1：一句话（≤25字）
- 步骤2：一段话（约5句）
- 步骤4：一页（约500-800字）
- 步骤5：每个角色一页
- 步骤6：四页（约2000-3000字）
- 步骤8：场景数量取决于小说长度（一般30-100个场景）
